# EventideV1

Eventide is a booking and provider discovery platform (backend: NestJS, frontend: Angular) used for demo and development purposes.

Quick start

1. Backend
   - cd backend
   - npm install
   - cp .env.example .env
   - npm run start:dev

2. Frontend
   - cd frontend
   - npm install
   - ng serve
